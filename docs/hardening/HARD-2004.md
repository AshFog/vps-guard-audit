# HARD-2004：清理或收紧防火墙规则

检测到“当前无人监听”的放行端口，不代表规则一定无用：服务可能暂时停止、按需启动或只在维护窗口使用。

工具只能生成候选列表，不能自动批量删除。用户应逐条确认端口、协议、来源网段和对应业务，并检查 Docker、面板、邮件、VPN 与备用 SSH 端口。

变更前保存 `ufw status numbered`、`ufw show raw` 和 `/etc/ufw`。每删除一条规则就验证相关服务，不提供“全部清理”选项。发生误阻断时，从控制台按备份重新添加 allow 规则。
