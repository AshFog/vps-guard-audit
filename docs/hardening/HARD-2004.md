# HARD-2004：清理或收紧防火墙规则

检测到“当前无人监听”的放行端口，不代表规则一定无用：服务可能暂时停止、按需启动或只在维护窗口使用。

工具不会根据“无人监听”自动删除。它显示 `ufw status numbered`，并且只处理用户明确输入的规则编号；用户仍应逐条确认端口、协议、来源网段和对应业务，并检查 Docker、面板、邮件、VPN 与备用 SSH 端口。

规则按编号倒序删除，避免前一条删除导致后续编号错位。变更后必须由第二 SSH 会话确认；未确认会按事务备份恢复 UFW 规则文件。
